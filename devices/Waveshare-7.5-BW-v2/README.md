# Deployment

* Copy the file ```include/config.h.example``` to ```include/config.h```
* Edit the ```config.h``` file and fill in the correct information
* Open this folder in VSCode (with platformio plugin) as a new project
* Build the project
* Upload the project
* ...
* \$\$\$
